# SMT Solver

## Installation Instructions

1. Make sure you have [anaconda](https://docs.anaconda.com/anaconda/install/index.html) installed. 
2. Navigate to the `Part A` folder and run `conda env create -f environment.yml` to create a conda environment with the required dependencies.
1. Activate the environment by running `conda activate gridworld`.

You can run the code by running `python main.py`. This will run the DQN on the GridWorld environment and perform some checking. You can also run `python main.py --help` to see the available options.

If you find any bugs, please let us know. This way, you can help us improve the assignment for next year's students.
